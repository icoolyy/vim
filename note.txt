基础操作：

ctrl + l 终端清屏

1、命令行输入vim默认进入normal模式，再输入命令符（例如：a、i、o）进入相关模式；
2、a\A表示在光标符\行最后那插入新字符；a append, A append affter line
3、i\I表示在光标符\行最后那插入新字符；i insert, I insert before line
4、o\O表示在当前行后\前添加新行；o open a line below, O open a line above
5、wq表示写入并推出操作。

vim4种模式

Nomal模式（ESC）
Insert模式（a,i,o,A,I,O)
Command模式
Visual模式（v,V）

Normal普通模式（浏览）
Esc插入模式回到普通模式
普通模式下进行各种命令操作和移动
大部分情况下，使用的是浏览

1、sp、vs可进行分屏编辑；
2、% s/str1/str2/[g]  可进行文本[全局]替换；
3、normal模式下，v/V进行可视化（光标/整行选中）操作；
4、y命令可以复制选中的块，p可以粘贴复制的块。

用vim自身的分屏模式

命令模式下 使用:sp 进行纵向分屏 使用:vsp进行横向分屏
使用ctrl+w+h/j/k/l 进行切换某个分屏

ctrl+h 删除字符
ctrl+w 删除单词
ctrl+u 删除一行
ctrl+b 前移一个字符
ctrl+f 后移一个字符
ctrl+c/[ 功能同Esc 切换insert和normal模式
gi 快速回到刚才编辑的地方
q:退出当前窗口
qa:退出所有

normal 模式下快速移动
h、j、k、l 左，下，上，右
:syntax on 语法高亮

0 到行首, ^ 行首第一个非空白字符
$ 行尾
数字 + gg 移动到某一行
w/W 移动到下个单词的开头 大小写区别,是否空白符分割
e/E 跳转到下一个单词结尾
b/B 移动到上一个单词的开头
***大小写区分自然单词和空白符强制分割的单词。
行间的搜索移动
f{char}移动到字符char上，t{char}移动到char的前一个字符，如果重复则可以用;/,进行下移
normal Mode
ctrf+f:下一屏
ctrl+b:上一屏
ctrl+d:向下翻半屏
ctrl+u:向上翻半屏

viw 选中当前word
daw : delete a word
dt{char}: delete to the char
dd : delete the row
x : delete a char
c : change
c : change a word
r/R 一次替换或一直替换
c/C c需要某种运动,cl与s基本相同,例如您可以键入cw删除整个单词并进入插入模式,C用您的编辑替换该行的其余部分（从光标位置开始）
s/S c删除一个字符并进入编辑模式,cc和S是同义词，用您的编辑替换整个行
.重复前一次操作
u撤销前次操作
ctrl+r撤销最近一次撤销
/或? 向后或向前搜索
n/N 定位到搜索项向后或向前移动
*/# 移动到新的位置继续匹配搜索,向前向后
